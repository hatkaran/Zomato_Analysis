# Numerical and Geospatial Analysis of Zomato Restaurant Data

Conducted numerical and geospatial analysis of Zomato restaurant data using Pandas, NumPy, SQLite, Geopy, and Folium. Performed exploratory data analysis on ratings, cuisines, and pricing trends, while leveraging geocoding and interactive heatmaps to visualize restaurant distribution and identify location-based patterns and market opportunities.
